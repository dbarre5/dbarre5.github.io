Conjugate Transfer Simulation of heat sink using OpenFOAM

July 2019

Overview

OpenFoam's conjugate heat transfer solver (chtMultiRegionFoam) is used to simulate heat transfer through a single heat sink fin. The simulation is compared against the analytical solution with decent results. A tutorial (in progress) for the case setup is provided for the case setup and scripts used to run this particular case.

Detailed Results

Analytical solution details + code, Grid independence study, Comparison to analytical solution, etc.
