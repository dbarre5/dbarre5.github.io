CAD Tennis Ball Launcher in Autodesk Fusion 360

Spring 2017

Overview

Tennis ball launcher prototype created in Fusion 360. I did this back in undergrad because Autodesk had a challenge where if you created an assembly of 15+ parts, you could get paid 250$. I submit it and got the money, which I was planning on using to actually build the tennis-launcher machine. However, as I was getting back into tennis, I was having wrist pains. Since I didn't have any plans on getting back into tennis, I never actually started building the machine (which I was going to do with 3D printing and an arduino).
